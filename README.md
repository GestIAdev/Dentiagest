##  Visión

Plataforma completa de gestión dental que combina:

- **Gestión Clínica Profesional** (16 módulos funcionales)
- **IA Generativa** (Selene Song Core)
- **Web3 Ecosystem** (DentalCoin + Patient Portal)
- **Compliance Enterprise** (GDPR + EU AI Act)

**Diferenciador**: Primera plataforma dental con IA generativa, blockchain y compliance integrados.

---

##  Arquitectura

### Stack Tecnológico

- **Frontend**: React 18 + TypeScript + Tailwind CSS
- **Backend**: FastAPI + Python 3.11 + SQLAlchemy
- **Database**: PostgreSQL 15 + Redis
- **AI Engine**: Selene Song Core (15K+ experiences)
- **Web3**: ethers.js + Sepolia testnet

### Estructura del Proyecto

\\\
frontend/          # React 18 SPA (14 páginas, 16+ componentes)
backend/           # FastAPI REST API (8 módulos)
patient-portal/    # PWA offline-first (6 módulos V3)
selene/            # IA autónoma (consciousness engine)
\\\

##  Instalación

### Prerrequisitos

- Node.js 18+
- Python 3.11+
- PostgreSQL 15+

### Quick Start

\\\ash
# Backend
cd backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
alembic upgrade head
uvicorn app.main:app --reload

# Frontend
cd frontend
npm install
npm run dev
\\\
